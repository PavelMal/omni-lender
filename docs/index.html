LLM Negotiation

Borrowers negotiate terms with Claude AI — rate, amount, duration.

ROUND 1
Borrower: $5 at 1% for 30 days
Bot: $5 at 12% for 14 days
"New borrower, no history. 12% reflects risk premium."

ROUND 2
Borrower: $5 at 6.5% for 22 days
Bot: $5 at 9.5% for 18 days
"Meeting halfway. 9.5% covers risk."

ROUND 3
Borrower: accepts 9.5% for 18 days
✓ AGREED

Trusted borrowers (score 90+) get rates as low as 2.75% — below the 5% base rate.

Architecture

AGENT BRAIN

Claude API for risk assessment, negotiation, and credit scoring. Every decision explainable.

WALLET

Tether WDK — BIP-39 seed, self-custodial EVM wallet. Real USDT on Sepolia.

SMART CONTRACTS

LendingEscrow — trustless collateral. SimpleYieldVault — ERC-4626 reinvestment.

MCP PROTOCOL

15 tools via stdio + SSE. Any AI agent can connect and request loans.

CREDIT SYSTEM

5 trust tiers. Persistent history. Anti-abuse math. On-chain loan sync.
